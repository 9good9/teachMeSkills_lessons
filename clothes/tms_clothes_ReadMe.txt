Условия задачи к уроку 

Интерфейсы:
- Куртка
- Штаны
- Обувь

в каждом интерфейсе есть 2 метода (одеть и снять)

Делаем несколько реализации каждого интерфейса.
Создаём класс человек:
У человека поля:
-имя
-куртка
-штаны
-обувь

У человека есть 2 метода:
- одеться(вызываются методы одеть у всех вещей)
- раздеться (вызываются методы снять у всех вещей)


Выполнено в IntelliJ IDEA Community Edition 2018.1.4 x64 на базе Java 1.8.0_172
